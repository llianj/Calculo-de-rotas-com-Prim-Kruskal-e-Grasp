# Calculo-de-rotas-com-Prim-Kruskal-e-Grasp
Algoritmo de cálculo de rotas em cidades que determina o menor caminho possível entre um ponto inicial, possíveis pontos de interesse e um ponto final. O algoritmo utiliza os algoritmos de grafos de Prim, Kruskal e Grasp

Para o uso do presente algoritmo, devemos primeiro iniciar o ambiente virtual
    Siga os seguintes passos no terminal:
    1 - Abra o terminal na pasta do projeto; ou abra directamente o terminal do VS Code do cmd 
    (somente funcionará no terminal do cmd do VS Code).
    2 - Digite no terminal ".\venv\Scripts\activate" - (sem aspas) e tecle Enter.
    3 - Após ativar o ambiente virtual, digite "streamlit run app_final.py" - (sem aspas) e tecle Enter.
    4 - O Streamlit abrirá automaticamente no navegador padrão, caso não abra, copie o link do terminal e cole no navegador.
    5 - Para desativar o ambiente virtual, digite "deactivate" - (sem aspas) e tecle Enter.

    OBS: Caso dê algum erro, é provavel que seja:
    * Alguma biblioteca que não foi instalada corretamente - Para resolver isso, instale as bibliotecas listadas no topo do código
    * Ambiente virtual não ativado corretamente - Verifique se o ambiente virtual está ativado conforme os passos acima, 
    caso não funcione crie um com o comando: "python -m venv .venv" - (sem aspas) e repita os passos acima na criação do ambiente virtual.

Iniciar o ambiente virtual:
#   .\.venv\Scripts\activate
#     streamlit run app_final.py

Desativar o ambiente virtual:
#   deactivate

Criar ambiente virtual:
#   python -m venv .venv
